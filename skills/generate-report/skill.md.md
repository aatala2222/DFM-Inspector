# Generate Report

## Description
Generates a formatted DFM inspection report summarizing findings, scores, and recommendations.

## Trigger
Use this skill when the user asks for a report, summary document, or formatted output of inspection results.

## Steps
1. Gather all inspection findings from the current session.
2. Apply the report template.
3. Include: Executive Summary, Findings Table, Risk Assessment, Recommendations.
4. Format as requested (table, document, or export).

## Example Prompts
- "Generate a DFM report for this part"
- "Create an inspection summary"
- "Export findings as a report"
