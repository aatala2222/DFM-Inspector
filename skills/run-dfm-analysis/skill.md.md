# Run DFM Analysis

## Description
Runs the full DFMInspector analysis pipeline on a design, applying all rules and generating a comprehensive assessment.

## Trigger
Use this skill when the user asks for a full analysis, complete DFM check, or comprehensive review.

## Steps
1. Identify the target design or part.
2. Run the DFMInspector analysis engine.
3. Apply all applicable process specifications.
4. Score the design on manufacturability (0-100).
5. Generate a prioritized list of issues with corrective actions.

## Example Prompts
- "Run a full DFM analysis on this part"
- "Give me a complete manufacturability assessment"
- "Score this design for manufacturing readiness"
